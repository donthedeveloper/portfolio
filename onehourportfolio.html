About The Project

This website was focused on showing my viewers how easy and quick it was build and launch their portfolio website using Bootstrap. My first live stream regarding this taught my viewers how to take an idea, and turn it into a website, starting with the discovery process. We explored a freelance client questionare and an online wireframing tool to help build a content-driven layout as a mockup. Then, we picked a color scheme (viewer's choice) and built this website using Bootstrap.
